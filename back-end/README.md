# F2 back-end

F2 back-end is strapi application.

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) or [yarn](<(https://pip.pypa.io/en/stable/)>) to install f2.

1. Clone the repo

```bash
git clone https://github.com/linhdatran3/Devera_Backend.git
```

2. Install package

```bash
npm install
```

OR

```bash
npm install
```
